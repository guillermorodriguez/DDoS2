# Project
Project to simulate the simulation of the data from a DDoS attack via a Linux VM.

# Run Instructions
Run the code through the python command line as provided below.
> python main.py

# Requirements
Python:     3.9.12
mininet:    2.3.0
pyvis=0.1.9

Installation Instructions
pip install pyvis

# Operating System
Ubuntu      16.04.6 LTS

# Output
The program outputs the network graphs for a denial of service attack node structure as well as a graph showing the basic network 
configuration.
